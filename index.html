Google Sheets API

Conecta tu aplicación con Google Sheets para sincronizar datos entre dispositivos. Copia el código de Apps Script al editor de tu hoja de cálculo y pega la URL aquí.

Instrucciones de configuración

1. Crea una hoja de cálculo en Google Sheets
2. Ve a Extensiones > Apps Script
3. Pega el código proporcionado
4. Implementa como aplicación web (acceso: Cualquiera)
5. Copia la URL generada y pégala arriba
6. Haz clic en "Sincronizar" en el encabezado para probar
